{\footnotesize\ttfamily int compare (\begin{DoxyParamCaption}\item[{const void $\ast$}]{a,  }\item[{const void $\ast$}]{b }\end{DoxyParamCaption})}



Compare two elements of the {\ttfamily side\+List} list, used for sorting the list. 

Sort side\+List array in order to retrieve the connectivity info more efficiently. Basically we want to sort the sides on three different levels\+: first, by their first node ID, secondly, by their second node ID, and thirdly by if they are a boundary condition side. The result might look something like this\+: 3 3 1 1 2 0 2 1 1 1 3 0 1 2 0 -\/-\/$>$ 2 1 1 3 1 0 2 2 1 2 2 1 3 3 0 3 3 0 3 3 1
